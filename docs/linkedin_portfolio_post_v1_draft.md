# LinkedIn / Portfolio Draft

I have been building TRACE, a local-first read-only IT Ops diagnostic console for support scenarios involving AD, DNS, SMB, endpoints, and Microsoft 365-style troubleshooting.

The first validated scenario is a FactoryOps file-share access diagnostic: given a user, share, host, domain, DNS server, and required group, TRACE checks the evidence path and produces a clear result with safe next steps. It does not change AD, DNS, permissions, firewall rules, or server configuration.

The goal is simple: make support investigations more evidence-based, repeatable, and safe.
